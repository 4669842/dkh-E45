&NewTask

Syntax    

[&NewTask subject = "<Subject>",owner = "<Owner>"]

Example Scripts

Description

Used to generate tasks in Outlook.

The task is not linked to any Schedule items within Visualfiles.

Parameters

amend

Amend the task item before sending.

end-date = <EndDate>

Sets the end date for the task. Unlike some of the other parameters, these values should not be enclosed in quotation marks. If no dates are entered, then the value defaults to today's date.

file = "<Attachment1,Attachment2, ...>"

Comma-separated list of attachments.

message = "<MessageText>"

Text for the body of the task.

owner = "<Owner>"

Recipient of  the task.

When this parameter is included in a NewTask command, the To section appears in the Outlook Task. This is because the task is assigned to someone else. However, if the Owner parameter is not included, the To section does not appear and the task is assigned to the current Outlook user.

save

Where specified in a script run within a case,  a copy of the task is saved into that case History.

start-date = <StartDate>

Set the start date for the task. Unlike some other parameters, these values should not be enclosed in quotation marks. If no dates are entered, then the value defaults to today's date.

subject = "<Subject>"

Text for the subject line of the task.

Example Scripts

1 Create a task

[&NewTask subject="Test",owner="Joe Bloggs",message="This is a new task"]

2 Specify a start and end date

[&NewTask subject="Test",owner="Joe Bloggs",message="This is a new task",start-date=05/04/2005,end-date=10/04/2005]

3 Attach a file

[&NewTask subject = "Test",owner = "Joe Bloggs",message = "This is a new task",start-date = 05/04/2005,end-date = 10/04/2005,File = "G:\users\J.Bloggs\Commands\&NewTask_command.htm"]

4 Amend a task

[&NewTask subject = "Test",owner = "Joe Bloggs",message = "This is a new task",start-date = 05/04/2005,end-date = 10/04/2005,File = "G:\users\J.Bloggs\Commands\&NewTask_command.htm",amend]

5 Save a task

[&NewTask subject = "Test",owner = "Joe Bloggs",message = "This is a new task",start-date=05/04/2005,end-date = 10/04/2005,File = "G:\users\J.Bloggs\Commands\&NewTask_command.htm",save]
