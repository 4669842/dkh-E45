&NewAppointment

Syntax    

[&NewAppointment subject="<Subject>",to="<Recipient>"]

Example Scripts

Description

Creates an email requesting an appointment and to send the email to all recipients in the recipient list.

Parameters

allday

Flags the appointment as an all day event.

If you do not specify start-date and end-date, the appointment will be for today.

If you specify an end-date without a start-date, the allday appointment will start and end on the end date.

If you specify a start-date but not the end-date, the allday appointment will start and end on the start date.

The start-time and end-time will be set to 00:00 for allday appointments.

amend

Amend the appointment before sending it.

category = "<Category1,Category2,...>"

Comma separated list of categories for the appointment.

The category of a new appointment can be specified from within Visualfiles so you can set a new appointment as Urgent so this is colour coded as urgent within Outlook.

This functionality is not standard in Outlook so in order to activate this functionality the following steps are required:

  • An additional registry setting is required to enable categories to be sent from Outlook:

Add ‘SendPersonalCategories’, with a DWord Value of 1 to allow (0 to block).

The registry path for to enter this setting varies depending on the version of Office you are running. For example for Office 2010:

HKEY_CURRENT_USER/Software/Microsoft/Office/14.0/Outlook/Preferences

Or,

HKEY_CURRENT_USER/Software/Policies/Microsoft/Office/14.0/Outlook/Preferences

Note: The \14.0\ entry in the path above is the Office 2010 version number installed on your local PC.

  • Remove the Outlook rule Clear categories on mail to allow new appointments to be received with categories attached.

Without both of these settings you will only be able to see categories on new appointments created locally.

end-date = <EndAppointmentDate>

Sets the end date for the appointment. Unlike some other parameters, these values should not be enclosed in quotation marks, for example: end-date = 22/08/03.
If no end-date is entered, the default value will be today's date.

end-time = "<TimeAppointmentEnds>"

Sets the end time for the appointment, in the format hh:mm. The start-time parameter defines the time the appointment will start. These values must be enclosed in quotation marks, for example, ,start-time = "13:00". If ,start-time is not defined, the value defaults to the current time.

Note: The Start-Time and End-Time are specified in a 24 hour format.
If the specified time is 9am or earlier, ensure that you enter the leading '0' e.g.  Start-Time=09:00.

Note: If the start-time or end-time parameters are not included, start-time defaults to the current time and end-time defaults to the current-time plus 30 minutes. As a result the appointment is created for the nearest appointment slot.

file = <Attachment1,Attachment2,...>

Comma-separated list of attachments.

location =

Enables the assignment of locations, such as meeting rooms, to an appointment. See examples 10 and 11.

message = "TextString"

Text for the body of the appointment message.

reminder = n

Gives you the option to set the reminder on or off, and to set the duration before the reminder in minutes.

n = the number of minutes before the start of the appointment to display the reminder.

If you do not add reminder=n then default Outlook 2007 behaviour applies.

Note: This feature applies to Outlook 2007 only, due to it having new function/API calls for setting reminders. Supplying the same functionality as Outlook appointments.

resource =

Enables the assignment of resources, such as projectors or phones, to an appointment. See examples 9 and 11.

save

Where specified in a script run within a case, a copy of the appointment will be saved into that file history.

start-date = <StartAppointmentDate>

Sets the start date for the appointment. Unlike some other parameters, these values should not be enclosed in quotation marks, for example, start-date = 22/08/03. If no start-date is entered, the default value will be today's date.

start-time = "<TimeAppointmentStarts>"

Sets the start time for the appointment, in the format hh:mm. The start-time parameter defines the time the appointment will start. These values should be enclosed in quotation marks, for example, start-time = "13:00". If start-time is not defined, the value defaults to the current time.

Note: The Start-Time and End-Time are specified in a 24 hour format.
If the specified time is 9am or earlier, ensure that you enter the leading '0' e.g.  Start-Time=09:00.

Note: If the start-time or end-time parameters are not included, start-time defaults to the current time and end-time defaults to the current-time plus 30 minutes. As a result the appointment is created for the nearest appointment slot.

subject = "<Subject>"

Text for the subject field of the appointment request email.

To="<Recipient>"

List of recipients: each separated by a semi-colon.

Example Scripts

1 Simple appointment request email

[&NewAppointment subject = "Test",To = "J.Bloggs@Visualfiles.com",Message = "Test Appointment"]

2 Specifying a start/end date and a start/end time

Note: The Start-Time and End-Time are specified in a 24 hour format.
If the specified time is 9am or earlier, ensure that you enter the leading '0' e.g.  Start-Time=09:00.

[&NewAppointment Subject = "Test",To = "J.Bloggs@visualfiles.com" ,Message = "Test Appointment" ,Start-Date = 05/04/2005 ,End-Date=05/04/2005 ,Start-Time = "15:00", End-Time = "15:30"]

3 Specifying a file attachment

[&NewAppointment Subject = "Test",To = "J.bloggs@visualfiles.com",Message = "Test Appointment",File = "G:\users\J.Bloggs\Commands\&NewTask_command.htm",Start-Date = 05/04/2005,End-Date = 05/04/2005,Start-Time = "15:00",End-Time = "15:30"]

4 Creating an all-day appointment

[&NewAppointment Subject = "Test",To = "J.bloggs@visualfiles.com",Message = "Test Appointment",File = "G:\users\J.Bloggs\Commands\&NewTask_command.htm",Start-Date = 05/04/2005,End-Date = 05/04/2005,allday]

5 Amending an appointment

[&NewAppointment Subject = "Test",To = "J.bloggs@visualfiles.com",Message = "Test Appointment",File = "G:\users\J.Bloggs\Commands\&NewTask_command.htm",Start-Date = 05/04/2005,End-Date = 05/04/2005,Start-Time = "15:00",End-Time = "15:30",amend]

6 Saving an appointment

[&NewAppointment Subject = "Test",To = "J.bloggs@visualfiles.com",Message = "Test Appointment",File = "G:\users\J.Bloggs\Commands\&NewTask_command.htm",Start-Date = 05/04/2005,End-Date = 05/04/2005,Start-Time = "15:00",End-Time = "15:30",save]

7 Sending to multiple recipients

[&NewAppointment Subject = "Test",To = "J.bloggs@visualfiles.com; J.Smith@visualfiles.com; J.Doe@visualfiles.com,Message = Test Appointment,File = G:\users\J.Bloggs\Commands\NewTask_command.htm,Start-Date = 05/04/2005,End-Date = 05/04/2005,Start-Time = 15:00,End-Time = 15:30,save]

8 Creating an appointment

Creating an appointment in Outlook via the COM Link.                                              

[&Assign REP12='Enter date of appointment'(date)]

[&Assign REP15='Enter time of appointment'(time)]

[&NewAppointment subject=CLI03+" - Appointment to Sign contract",to=GCF05+"@solicitorname.co.uk",message=CLI03+" - Appointment to Sign contract",start-date=REP12,end-date=REP12,start-time=REP15,end-time=REP15+.30,amend]

Setting the end time for an appointment

In the script above REP15 is set as a time field. The number after the time field +.30 is interpreted as a time value and added to the value in the time field.

The internal code converts the number before the decimal point to hours and after the decimal point to minutes.

Note: In order to use a time database field (ABC01 = 8:00) to populate Start-Time=ABC01, the time must be in 24 hour format.
If the specified time is 9am or earlier, ensure that you enter the leading '0' e.g.  Start-Time=09:00.

9 Assign a resource to an appointment

To assign a camera and a projector to an appointment:

[&NewAppointment to="a.person@business.co.uk", subject = "test", message="test", resource="camera1@business.co.uk;projector2@business.co.uk", start-date="25/12/08", end-date="25/12/08", start-time="04:30", end-time="05:16", amend]

10 Assign a location to an appointment

To assign a meeting room to an appointment:

[&NewAppointment to="a.person@business.co.uk", subject = "test", message="test", location="meetingroom3@business.co.uk", start-date="25/12/08", end-date="25/12/08", start-time="04:30", end-time="05:16", amend]

11 Assign a resource and a location to an appointment

To assign a resource (camera and projector) and a meeting room to an appointment:

[&NewAppointment to="a.person@business.co.uk", subject = "test", message="test", resource="camera1@business.co.uk;projector2@business.co.uk", location="meetingroom3@business.co.uk", start-date="25/12/08", end-date="25/12/08", start-time="04:30", end-time="05:16", amend]
