&NewDateTime

Syntax    

[{&NewDateTime date=} ]

Example Scripts

Description

Calculates a new date and time by applying a time period to a date and time. This command enables a period of time to be added to a date and time.

All parameters except date= are optional.

Notes

date-expr: An expression that resolves to a valid date value.

time-expr: An expression that resolves to a valid time value.

integer-expr: An expression that resolves to a whole number.

field-expr: An expression that resolves to the name of a temporary or database field.

Parameters

date= <date-expr>

Date to add time interval to.

days= <integer-expression>

Specifies number of days to add to date and time.

dayslast

Largest time interval types will be added first.

hours= <integer-expression>

Specifies number of hours to add to date and time.

minutes= <integer-expression>

Specifies number of minutes to add to date and time.

months= <integer-expression>

Specifies number of months to add to date and time.

newdate= <field-expression>

Field to place resulting date into.

newtime= <field-expression>

Field to place resulting time into.

time= <time-expr>

Is time to add time interval to (if not specified defaults to "00:00").

weeks= <integer-expression>

Specifies number of weeks to add to date and time.

workingdays=

Adds the specified number of working days.

work-date

Used to ensure the result is a working day. If the result of &NewDateTime falls on a non-working day the result are adjusted to the last working day before the result date.

years= <integer-expression>

Specifies number of years to add to date and time.

Example Scripts

1 months, days, newdate paramaters

[&Assign LF01="30/06/2010"]

[&NewDateTime date=LF01, months=1, days=1, newdate="LF02"]

[&Message LF02]

Displays: 01/08/2010

2 months, days, newdate, dayslast parameters

[&Assign LF01="30/06/2010"]

[&NewDateTime date=LF01, months=1, days=1, newdate="LF02", dayslast]

[&Message LF02]

Displays: 31/07/2010

3 time, hours, newdate, newtime parameters

[&Assign LF01="01/01/2010"]

[&Assign LF02="13:30"]

[&NewDateTime date=LF01, time=LF02, hours=12, newdate="LF03", newtime="LF04"]

[&Message LF03 + " at " + LF04]

Displays: 02/01/2010 at 1:30
