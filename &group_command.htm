&Group

Syntax    

[&Group ]

Example Scripts

Description

Manipulates particular instances of a group used within the &Grid command.

Parameters

action = "<Getindex>"

Gets an index for the group; there can only be one occurrence of this parameter.
This parameter is used to return the group iteration number of a specific screen; essentially a group iteration identifier used in conjunction with the Setindex parameter.
Getindex acquires the number and then Setindex uses this number to lock onto the screen, enabling the developer to relate to that specific group screen.
These parameters provide a speedy method of locating the required group iteration by removing the need to use &Foreach to find the one they want.

action = "<Setindex>"

Sets the index for the group; there can only be one occurrence of this parameter. For more details see action = "<Getindex>" above.

code = "<IdentifyingCode>"

Enter a valid group code.

index = <IndexField>

Used with the setindex option (action parameter) to identify the group iteration.

result = <FieldToStoreResult>

Used with getindex option (action parameter) to return the current index.

Example Scripts

1 Formats groups within a grid

[&Assign TF04  =  0]

[&Assign TF01  =  0]

[&Assign TF02  =  ""]

[&Assign TF03  =  ""]

[&Grid action  =  "create"]

[&Grid action  =  "create-col",name  =  "Team member",format  =  "X(60)",type  =  "char"]

[&Grid action  =  "create-col",name  =  "Role",format  =  "X(60)",type  =  "char"]

[&Grid action  =  "create-col",name  =  "ID",format=">>>999.99",type  =  "deci",key,hidden]

[&Foreach TEAM]

   [&Assign TF04  =  TF04+1]

   [&Grid action  =  "create-row"]

   [&Assign GR01  =  TEM01]

   [&Assign GR02 = TEM02]

   [&Group code = "TEAM",action = "getindex",result = TF01]

   [&Assign GR03 = TF01]

   [&Assign TF01 = 0]

[&Endfor]

[&If TF04 < 25]

  [&Assign TF04 = TF04+1]

[&Else]

  [&Assign TF04 = 25]

[&Endif]

[&Grid action = "display",title = "Team Members",button1 = "Cancel",button2 = "Open",button3 = "Select",result = TF01,rows = TF04,response = TF02,default = "3"]

[&If TF02 = "button3" &or TF02 = "go"]

   [&Group code = "TEAM",action = "setindex",index = TF01]

   [&Show TEM]

[&Endif]

[&Grid action = "delete"]

2 Linked case prefix

[&Group code="a_ABCD", action="getindex", result=TF01]

3 Getindex parameter, select an entry

Use the Getindex parameter to select an entry when using a grid:

[&Call FUN-DEFENDANTGRID]

*** Display the Grid

[&Grid action="Display", title="Defendant / Respondent Details (Double-click to edit details)", button1="Exit", button2="Select", button3="New Defendant", button4="Delete", response=LF01,rows=10, sort="SortOrder"]

[&If LF01 &IsOneOf "Go,Button2"] * Select one of the items

[&Group code="GDEF", action="setindex",index=GR04]

[&Display DEF, group="GDEF"]

[&Else]

[&If LF01 = "Button3"] * Add a new other party

[&Select GDEF:first]

[&If OTR01<>""]

[&Select OTPS:new]

[&Count type = "GROUP",code = "GDEF",result = LF01]

[&Assign DEF04=LF01]

[&EndIf]

[&Display DEF, group="GDEF"]

[&Else]

[&If LF01="button4"]

[&Group code="GDEF", action="setindex",index=GR04]

[&Assign LF10="Are you sure you want to remove "+DEF03+" as a contact from this file?"]

[&If prompt(LF10)(question)="Yes"]

[&Delete GDEF]

[&EndIf]

[&EndIf]

[&EndIf]

[&EndIf]
