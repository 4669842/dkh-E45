&ProgressBar

Syntax    

[{&ProgressBar mode=}]

[{"ON"|"SET"|"OFF"}]

Example Scripts

Description

This command enables developers to display a progress bar when a script is running indicating the of progress of the script and controlling the text displayed in the progress bar's popup frame.

Note: This command is not support in Visualfiles Browser.

The scripted progress bar appears as follows e.g.:

Note: See example 5 for more information on this example.

Note: The &ProgressBar command does not work when used with the &Show command. This is a limitation imposed by Windows. The progress bar  screen is a window (non-modal) object, whilst the screen shown using the &Show command is a dialog box (modal) object, and it is not possible to display a window on top of a dialog box.

Parameters

"OFF"

Switch off the progress bar using "OFF": . See example 1.

"ON"

Show the progress bar using "ON": . See example 2.

"SET"

Set the text (mode="SET") that appears inside the popup frame and the title bar of the frame using syntax text= and title=.

Image=

Set the image that appears inside the popup frame. See example 4.

Note: This parameter supports .ico images, which need to located be in your LNimg folder.

Text=

Set the text that appears inside the popup frame.

Title=

Set the text that appears inside the title bar of the frame

Example Scripts

1 Switch off progress bar

[&ProgressBar mode="OFF"]

2 Display progress bar

[&ProgressBar mode="ON"]

3 Set the text and title bar display in popup frames; text= and title=

Set the text (mode="SET") that appears inside the popup frame and the title bar of the frame using the parameters text= and title=.

[&ProgressBar mode="ON", text="Stage 1, please wait", title="Visualfiles ODPP"]

[some other stuff]

[&ProgressBar mode="SET", text="Stage 2, please wait", title="Visualfiles ODPP"]

[some more stuff]

[&ProgressBar mode="SET", text="Stage 3, please wait", title="Visualfiles ODPP"]

[some final stuff]

[&ProgressBar mode="OFF"]

4 Add an icon image to the progress bar; image=

[&ProgressBar Mode="On", image="vfile.ico", title="hi", text="how are you?"]

[&Message "pause"]

[&ProgressBar Mode="OFF", title="hi", text="bye"]

Note: The .ico image needs to be in your LNimg folder.
If you are using both the solbmp and lnimg image locations, you need to label the image accordingly because they are both contained in the program directory.
Icon from the solbmp folder:
Image = "Solbmp/vfile.ico"
Icon from the lnimg folder:
Image = "Lnimg/vfile.ico"

5 Sample &ProgressBar script

When the script below is run the progress bar is displayed with the script name in the title bar:

**********************************************************************

***  Script: HISTORYADD                                            ***

***  Called By:                                                    ***

***  Calls:                                                        ***

***  Created By: VF On 24/12/10                                    ***

**********************************************************************

[&Assign LF02=1]

[&Assign LF03='enter limit'(number)]

[&ProgressBar mode="ON", text="Add items to limit of:"+LF03, title="HistoryAdd"]

[&DoWhile LF02<=LF03]

&History "Document:"+LF02+" matter take on from previous law firm"

[&ProgressBar mode="ON", text="Added item:"+LF02, title="HistoryAdd"]

[&Assign LF02=LF02+1]

[&EndDoWhile]

[&ProgressBar mode="OFF"]
