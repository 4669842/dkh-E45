&AccWarnings

Syntax    

[&AccWarnings OFF|ON]

Description

The &AccWarnings command is used to stop the SOS Practice Manager link from displaying alert messages when Practice Manager returns an error status to Visualfiles.

[&AccWarnings OFF] Stops warning messages appearing.

[&AccWarnings ON] Reenables the display of warning messages.

Note: The display of warning messages always resumes at the end of a script run that specifies [&AccWarnings OFF] regardless of whether [&AccWarnings ON] is used, providing the script was not called from another script where warning messages were off.

Example Scripts

1 Stop warning messages appearing

[&AccWarnings OFF]

2 Display of warning messages

[&AccWarnings ON]
