&MissPay

Syntax    

[&MissPay]

Description

Updates the arrears field in a debt file where a payment has not been made, for example, where today's date is later than the Next Payment date for the file.

&MissPay can be run from a script or from the Missed Payment button on the Payment Arrangement dialog box.

There are no parameters for this command.

Example Script

1 Add a missed payment to arrears field

This script asks the user if a payment has been received. Where Yes is selected, the message 'Please enter payment details' is displayed and the Payments dialog box is displayed for the user to enter the details of the payment.

Where the user selects No, the &MissPay command will add the missed payment amount to the arrears field of the Payment Arrangement dialog box. PY is the Payment code and PY:last is the last payment made in the in the Payment group screen.

[&If 'Has a payment been received?'(Question) = "Yes"]
[&Message "Please enter payment details"]
[&Show PY]
[&Select PY:last]
[&Else]
[&MissPay]
