&Load

Syntax    

[&Load Type = "Bill", Bill-Num = "BillNumber"]

[&Load Type = "History"]

Example Scripts

Description

Loads a bill into a temp table to be accessed using BI, BG or BT field codes.  This speeds up the creation of a bill and provides better filtering and totalling abilities.  A Bill-Num must be specified to indicate which bill the temporary bill table should be loaded for.

&Load should always be followed by the &Unload command.

BI, BG and BT codes are used to collect values for calculation and manipulation:

  • BI = Each individual History and Ledger item on a bill.

  • BG = Items which have been grouped: for example, the user can total all history items (total all BI codes that represent history items).

  • BT = Represent Bill total fields: adds up all the Net and VAT totals in all BI fields.

Note: The information on this page is only applicable to systems with an activated Billing Module.

Note: This command is not supported in Visualfiles Browser.

Parameters

Bill-Num = "<BillNumber>"

Selects which bill case to load.

Display

Displays the History Document selection screen. When this parameter is not used all history items are automatically returned.

Mt-Int-code

Used to select which cases history to load.

Multi

Used to select multiple history documents.

Type =

Used to select either:

Type = "Bill"

Selects the temporary table of Bill to load the billing details.

Type = "History"

Loads the file History. &Load type=history lets you select a number of history items for a chosen matter. These are then placed in a grid so that users can carry out a FOREACH GR. The history number is added to the returned field list.

GR1 = Description

GR2 = Document ID

GR3 = Author

GR4 = Document ID and Date

Example Scripts

1 Loading the bill table for a specific bill

This code will load the bill temporary table for the bill case with the internal code (MT30) of 145. The user will then be provided with the ability to interrogate the bill codes of BI, BG and BT.

[&Load Type="Bill", Bill-Num="145"]
