&FieldValue

Syntax    

[&FieldValue

Description

The &FieldValue command enables you to indirectly assign values to database fields.

Parameters

<field-name-expr>

An expression that evaluates to a Visualfiles field code. Any trailing .00 in the resolved expression value is ignored.

<value-expr>

An expression that resolves to the value to be assigned to the field code specified by <field-name-expr>.

Example Scripts

1 Assign a value to a field

Assign the value "Visualfiles" to field ABC01.

[&Assign LF01="ABC01"]

[&FieldValue LF01="Visualfiles"]

2 Generic script to blank text fields

[&Call "SYSTEM.CLEAR-FIELDS", input="ABC01 ABC02 ABC03 BCD01 BCD04 BCD05"]

CLEAR-FIELDS script:

[&Assign LF01=0]

[&Dowhile LF01 < length(PI01,delimiter=" ")]

  [&Assign LF01=LF01+1]

  [&FieldValue word(PI01, first=LF01)=""]

[&EndDowhile]
