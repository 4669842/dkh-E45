&Page

Syntax    

[{&Page <WebPageCode>}]

Example Scripts

Description

&Page is only used with SolCase Online web pages. The command is used to validate/check data entered by the SolCase Online user, often as part of a conditional statement. When included in the Page Submit script, &Page controls the navigation of the end user.

The &Page command specifies which pages a SolCase Online user navigates to next. It is possible to validate and check the data entered by the web user and navigate the web user to a range of online web pages based on that data.

&Page takes precedence over the page specified as the Forward Link.

Parameter

<WebPageCode>

Name of the page to display.

Example Scripts

1 Navigate the user to the Case Summary

[&Page Summary]

2 Page Submit script

This code is from a Page Submit script.  Where a user responds Yes in a text field on a web page to indicate they have completed all the necessary details, they are directed to the web page Complete after selecting Submit. Failing to complete TF01 or entering a value other than Yes will direct the user to the page Details.

[&If TF01="yes"]

   [&Page Complete]

[&Else]

   [&Page Details]

[&Endif]
