&Field

Syntax    

[&Field code = "{<DBScreenCode>"|"<DBFieldCode>"},attribute = "<Attribute>",value = "<ValueString>"] 

Example Scripts

Description

This command is used to manipulate database fields on a screen.

&Field can be used in a View Screen script, a field and a button.

Parameters

attribute = <Attribute>

Specifies the attribute  to be manipulated:

Backcolour

Forecolour

Hidden

Image: Used in conjunction with the value parameter to specify an image for a Static Image field type (Property type), see example 8.

Label-Backcolour

Label-Forecolour

Label-Visible

List-Items: Enables you to create a new selection list for a screen field. See example 7.

List-Items-Append: Enables you to add a number of items to an existing selection list.

Navigate: Used in conjunction with the value parameter to manipulate the properties of a web page field type.

Read-only

Tooltip

Vertical-scrollbar: Used to show or hide the vertical scrollbar in an embedded grid, see example 10.

Visible: Used to hide embedded grid screen fields, see example 9.

code = <{DBScreenCode|DBFieldCode}>

Specify a database screen code to change all fields on the screen or a database field code to change only that field.

The code parameter may be expressed as the screen code plus the expression TEXT or RECT plus the field number to adjust the properties of screen text or rectangles respectively, for example, ABCTEXT02 is the screen code ABC plus the expression TEXT and field code 02.

Example Scripts

1 Modify all the fields in the screen ABC, to have a yellow background colour

[&Field code = "ABC",attribute = "backcolour",value = "14"]

2 Hide the database field ABC04

[&Field code = "ABC04",attribute = "visible",value = "No"]

3 Create red text on ABC06 and make it read-only

[&Field code = "ABC06",attribute = "forecolour,read-only",value = "12,yes"]

4 Modify ABCTEXT02  to have a blue background

[&Field code = "ABCTEXT02",attribute = "backcolour",value = "9"]

5 Modify Rectangle 03 on ABCRECT03, to have a green background

[&Field code = "ABCRECT03",attribute = "backcolour",value = "10"]

6 Create navigation buttons for a web page field type

[&Field code="web01",Attribute="navigate",Value="BACK"]

[&Field code="web01",Attribute="navigate",Value="FORWARD"]

[&Field code="web01",Attribute="navigate",Value="REFRESH"]

[&Field code="web01",Attribute="navigate",Value="HOME"]

[&Field code="web01",Attribute="navigate",Value="http://www.visualfiles.com/"]

7 List-Items parameter

[&Field code="ABC01", attribute="List-Items", value="One;Two;Three;Four;Five"]

The value must be semi-colon delimited.

This command is added to the view script that runs when screen ABC is displayed and results in field 01 having a new selection list.

It is also possible to set up the selection list using temporary fields and user-defined screen fields, for example:

[&Assign LF01="Red;Green;White"]

[&Field code=ABC01, attribute=List-Items, value=LF01 + ; + AAA03]

Note: The List-Items parameter and the List-Items-Agenda parameter only work when they are used from the view script, and they can only be applied to a field which already has a selection list applied to it in its field properties.

8 Specify an image for a screen

[&Field code="RAA05", attribute="image", value="lnimg\letter.jpg"]

The image file must be held in your LNimg folder, and the field type needs to be a static image.

9 Hide the embedded grid field

To hide the embedded grid field, where TST01 in the script below is the embedded grid:

[&Field code = "TST01",attribute = "visible",value = "No"]

10 Vertical-scrollbar

Set the vertical scrollbar attribute of a grid to yes or no.

[&Field code="ABC01", attribute="vertical-scrollbar", value="no"]

11 label and script attributes

To change the field label on the first screen field on screen ABC use the following syntax:

[&Field code= "abc01" ,attribute= "label" ,value= "New field label here" ]

To change the script attached to the field label use the following syntax:

[&Field code= "abc01" ,attribute= "script" ,value= "script name here" ]

Note: The script name (code) needs to be entered, not the full description.

12 text and script attributes

To change the text displayed on the first screen text field on screen ABC use the following syntax:

[&Field code= "abctext01" ,attribute= "text" ,value= "New screen text here" ]

To change the script attached to the screen text on the same field the  syntax would be:

[&Field code= "abctext01" ,attribute= "script" ,value= "script name here" ]

Note: The script name (code) needs to be entered, not the full description.

Developers' Tip

There must be a matching value for each attribute specified and the values must be separated by commas.
